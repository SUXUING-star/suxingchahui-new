# 关于项目开发
本仓库只包含前端的逻辑

以下是.env.example
```
# VITE_R2_PUBLIC_URL = R2存储库/其他oss服务

# VITE_API_URL = 后端API接口路径，后端域名/api
```

拉取项目

```
git clone https://github.com/SUXUING-star/suxingchahui-new.git
```

```
npm install # 安装npm包
npm run dev # 开启开发服务器
```
